Початковий код: https://codepen.io/anon/pen/gZJgLg

Задача: написати простий редактор тексту.

- Додайте 3 кнопки для зміни кольору, розміру шрифту та зміни фонового кольору тексту.

- Додайте можливість створювати перенос тексту в новий рядок. Повинен створюватись новий <br/>.

- Початок та завершення слова текстового блоку з фоном повинні мати радіус.

- Створіть можливість переводити текст в json-формат. Масив повинен містити властивості: текстовий блок, його властивості: розмір, колір, фон. Основна вимога: коли користувач виділяє якийсь фрагмент текст і змінює його властивість (колір, розмір, фон) - масив має перекалькулюватись так, щоб суміжні блоки з однаковими атрибутами не дублювались. Приклад:

правильно:


{

   {

      text: 'My lovely',

      fontSize: '12px',

      color: 'red'

   },

   {

      text: 'little',

      fontSize: '16px',

      color: 'pink'

   },

}


неправильно:

{

   {

      text: 'My ',

      fontSize: '12px',

      color: 'red'

   },

   {

      text: 'lovely',

      fontSize: '12px',

      color: 'red'

   },

   {

      text: 'little',

      fontSize: '16px',

      color: 'pink'

   },

}
